# Rekono Logo Pack

This folder contains enterprise-grade SVG logo assets for Rekono with a fintech visual direction.

## Recommended Primary (Handcrafted)

- rekono-master-logo.svg
- rekono-master-icon.svg
- rekono-master-logo-mono-black.svg
- rekono-master-icon-mono-black.svg
- rekono-master-logo-mono-white.svg
- rekono-master-icon-mono-white.svg

These are the only approved brand master assets in this folder.

## Design Characteristics

- Abstract "Monolith/Core" geometry (Layered Shards)
- Deep precision gradients indicating premium technology
- Flat design core with soft depth-shadows (removable for monochrome)
- Scales flawlessly to app icon size or billboard size
- Completely custom, non-generic enterprise insignia

## Usage Guidance

- Use the master icon and logo variants everywhere product branding is needed.
- Keep the older non-master logo files out of new work.
